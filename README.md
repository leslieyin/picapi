[https://random-picture.vercel.app/](https://random-picture.vercel.app/)_(部署到 vercel)_

### 注意:

| 路径            | 说明                                   |
| --------------- | -------------------------------------- |
| ./bing.csv       | 务必一行一个 url，不要输入不完整的 url |
